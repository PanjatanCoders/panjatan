# Container Shipment Data UI

A web application for exploring, filtering, sorting, and downloading container shipment records. Built with vanilla JavaScript, HTML, and Tailwind CSS.

## Features

- Interactive table with search, filter, sort, and pagination
- Download filtered data as CSV, Excel, or TXT
- Responsive and modern UI

## Getting Started

1. Clone the repository.
2. Open `table.html` in your browser.

No build step or server required.

## File Structure

- `table.html` — Main UI page
- `table-script.js` — Table logic and interactivity
- `style.css` — Custom styles
- `icon.png` — Favicon

## Download Options

- **CSV**: Download as comma-separated values
- **Excel**: Download as `.xls` (basic HTML table format)
- **TXT**: Download as tab-separated text
